Run
---

Install Atom:  
`./scripts/install-atom.sh`

Archive website:  
`./scripts/archive-website.sh <WEBSITE>`

Archive url:  
`./scripts/archive-url.sh <URL>`

Flush DNS (Ubuntu):  
`./scripts/flush-dns.sh`

Website IPv6, Diode and SSL certificate validation:  
`./scripts/epfl-net-test.sh -h <HOST>`
